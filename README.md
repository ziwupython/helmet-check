# helmet-check
基于YOLO的安全帽佩戴检测系统~Python+算法模型+目标检测+2026原创

## 一、介绍
安全帽佩戴检测系统，技术栈如下：
● 前端：Vue3+Element plus
● 后端：Flask框架
● 算法：YOLO
关键技术：YOLO模型、模型训练、目标检测、人工智能、Python
